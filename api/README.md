# Application de gestion bâtiment - Spring Boot

## Présentation

Application web de gestion pour une entreprise de plomberie et chauffage.

Elle permet de gérer :

- les clients
- les chantiers
- les contrats

L'application contient également un espace administrateur.

---

## Technologies utilisées

- Java
- Spring Boot
- Thymeleaf
- MySQL
- Maven
- Docker
- Git / GitHub

---

## Fonctionnalités

### Interface publique

- page d’accueil
- présentation de l’entreprise

### Interface administrateur

- tableau de bord
- gestion des clients
- gestion des chantiers
- création de contrats
- impression du contrat

---

## Lancer le projet

Dans le dossier du projet :

```bash
mvn spring-boot:run
```
